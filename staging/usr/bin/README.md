## The Naive binary

Download the `openwrt-aarch64_generic-static` kind from [Naive Github Releases](https://github.com/klzgrad/naiveproxy/releases/latest) inside this directory and name the binary `naive`.

## The Sing-box binary

Download the `openwrt_aarch64_generic` kind from [Sing-box Github Releases](https://github.com/SagerNet/sing-box/releases/latest) inside this directory and name the binary `sing-box`.
