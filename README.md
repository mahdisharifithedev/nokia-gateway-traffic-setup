## My personal network traffic routing setup for my Nokia Fastmile 3.2 Gateway 5G

- Follow [binary download](staging/usr/bin/README.md).
- Run `package.sh` locally to package the required assets and serve the compressed file on the network.
- Run `load.sh` through `ssh hguser@192.168.1.1` (switch to root) to download and install everything.
- Reboot and test it out.
